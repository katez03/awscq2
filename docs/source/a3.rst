Follow these initial steps to access the A3 challenge:

#. In the **ASSIGNMENT** section, select **ROUTE AND GO** to begin Assignment 3.

   .. image:: pictures/0003-A3.png

      :align: center
      :width: 600px

#. Select **GO TO SOLUTION CENTER** to view the lab steps.

      :align: center
      :width: 600px
